# Scripts Directory

This directory contains utility scripts for database migrations and maintenance.

## Available Scripts

### Migration Scripts

#### `migrate-analytics.ts`
Migrates analytics tables to the database.

**Usage**:
```bash
npx ts-node scripts/migrate-analytics.ts
```
