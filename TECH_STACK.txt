  ARCHITECTURE PATTERNS & DESIGN
================================================================
  - RAG (Retrieval-Augmented Generation) — core AI pattern
  - Multi-agent architecture (configurable named AI agents)
  - REST API (Express, versioned at /api/v1)
  - Feature-based folder structure (backend/src/features/*)
  - Two-tier caching (L1 in-memory + L2 Redis)
  - Decorator-based validation (class-validator, reflect-metadata)
  - DTO pattern (Data Transfer Objects)
  - Repository/Service/Controller separation
  - Background job processing (BullMQ training queue)
  - WebSocket-based real-time updates (Socket.IO)
  - Graceful shutdown handling
  - JWT-based stateless auth
